This application is designed to simplify the process of completing BPKB, STNK and FISIK forms. Data is entered via a single application and the output consists of these forms, for which templates are provided in the /templates folder.

The output files must adhere to the provided templates as they will be printed and used as physical documents.

Flow

1. Open the application
2. Enter data according to the form
3. Click Generate
4. Select a folder to save the completed documents
5. Process complete
6. Open the results in the output folder

Templates are available in the /templates folder in .docx format
